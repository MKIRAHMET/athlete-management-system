# Athlete Management System

PHP/MySQL athlete and event management website with an admin dashboard.

## Features
- Athlete profiles
- Event listings
- Coach/admin login
- Booking/request management
- Dynamic PHP pages
- MySQL database integration

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Security Note
Database credentials are loaded through environment variables and are not included in the repository.

## How to Run
1. Clone the repository
2. Create a MySQL database
3. Import the SQL file if available
4. Configure database environment variables
5. Run the project locally with XAMPP/MAMP or a PHP server
